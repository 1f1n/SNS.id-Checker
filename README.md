<p align="center">
	Easily check SNS.id for available and unregistered Solana domain names.
</p><br>
<h1 align="left">
Setup
</h1>

`
pip install -r requirements.txt
`
<br><br>
`
Add usernames to domains.txt
`
<br><br>
`
python main.py
`
